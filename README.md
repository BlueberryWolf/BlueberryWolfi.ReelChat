# BlueberryWolfi.ReelChat
Proximity Voice Chat Mod for WEBFISHING using GDWeave

I am too exhausted to make a proper usage docs but
default keybinds are:
F: toggle mute
T: push to talk

You can change keybinds in the mod settings
